What is the Document Object Model (DOM)?

The DOM is a programming interface for web documents. It represents the page so that programs can change the document structure, style, and content.

How do you select an element from the DOM?

You can select elements using methods like document.getElementById, document.getElementsByClassName, and document.querySelector.

How do you manipulate an element's attributes and styles using the DOM?

You can manipulate an element's attributes using methods like setAttribute and getAttribute. For styles, you can modify the style property directly or use classList to add or remove classes.
